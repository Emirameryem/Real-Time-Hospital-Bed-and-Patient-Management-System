This project is a real-time hospital bed and patient management system designed to digitalize and optimize hospital workflows.
It enables tracking of bed status, patient assignments, and cleaning processes through a centralized platform.
The system is built with .NET 8 Web API and SQL Server, supports JWT-based authentication and role-based access control (RBAC), and uses asynchronous communication for high performance.
Comprehensive logging mechanisms provide transparency and traceability in hospital operations.

Bu proje, hastane iş süreçlerini dijitalleştirmek ve optimize etmek amacıyla geliştirilen gerçek zamanlı bir hasta ve yatak yönetim sistemidir.
Yatak durumları, hasta yerleşimleri ve temizlik süreçleri merkezi bir platform üzerinden takip edilebilmektedir.
Sistem .NET 8 Web API ve SQL Server kullanılarak geliştirilmiş, JWT tabanlı kimlik doğrulama ve rol bazlı yetkilendirme (RBAC) ile güvenli hale getirilmiştir.
Asenkron API iletişimi ve kapsamlı loglama mekanizmaları sayesinde yüksek performans, güvenilirlik ve izlenebilirlik sağlanmıştır.
